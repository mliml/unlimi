# Hero Section 图片说明

请将落地页主视觉图片放在此目录下。

## 文件要求

- **文件名：** `hero-main.png`
- **推荐尺寸：** 1920×1080px 或 16:10 比例
- **支持格式：** PNG、JPG、WebP
- **文件大小：** 建议优化后 < 500KB，确保快速加载

## 在代码中的使用

图片会自动加载，路径为：
```
/images/hero/hero-main.png
```

## 占位符行为

如果图片不存在，页面会显示一个占位符，包含：
- 图片图标
- "产品展示图" 文字
- 使用说明

一旦您将图片放在正确位置，页面会自动显示实际图片。

## 图片优化建议

1. 使用图片压缩工具（如 TinyPNG）减小文件大小
2. 如果使用 PNG，考虑转换为 WebP 格式以获得更好的压缩率
3. 确保图片清晰度适合高分辨率屏幕（Retina）
4. 图片内容应突出 UnLimi 产品特色

## 示例图片内容

建议图片展示：
- UnLimi 应用界面截图
- 心理咨询对话场景
- 产品核心功能演示
- 温馨、专业的视觉氛围
